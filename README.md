# Geo Diver X
Group project made with unreal engine 5, goal is to gain points by breaking the blocks and defeating the enemies by countering their bullets. Survive!

## How to play

- download the rar, unpack and run "UnrealSCHMUP.exe"

### Controls

- Arrowkeys        : Move
- Space[click]     : Counter
- Space[Hold]      : Drill
- G                : Boost 
- Esc              : Menu
